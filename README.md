# Wolfhunters

Wolfhunt

En Wolfhunt eres una cazadora que vive cerca de una montaña en Alaska, tras un encuentro desafortunado con un lobo negro en el bosque deberás acudir al pueblo a reparar tus herramientas y volver a ponerte de pie para enfrentar a la bestia que te desafío y ganarle de una vez por todas.

Mecánicas:
- Caza
- Recolección
- Trading

Controles:
- Teclado: WASD para moverse, espacio para saltar, click izquierdo para atacar, click derecho para apuntar, E para interactuar, C para cambiar de camara en primera y tercera persona, SHIFT para correr, rueda de mouse para cambiar de arma.
